But du jeu

Case défis : le joueur relève un mini défi pour avancer plus rapidement.

Bleu : Si la question est répondue correctement, le joueur avance de 1 case.

Blanc : Si la question est répondue correctement, le joueur avance de 2 cases.

Rouge : Si la question est répondue correctement, le joueur avance de 3 cases.

Pour y arriver, il faut répondre à des questions plus ou moins difficiles selon votre choix.

3 niveaux de difficulté que le joueur choisit au préalable :

Bleu : facile

Blanc : intermédiaire

Rouge : difficile

Parcourir le plateau le plus rapidement possible.

Déroulement du jeu

Mise en place : Chaque joueur place son pion au départ. Le joueur qui commence est celui qui obtient le plus grand nombre avec le dé.

Tour de jeu :

Le joueur lance le dé et avance son pion du nombre de cases indiqué.

Cases Questions : Si le joueur tombe sur une case “Question”, il doit répondre à une question tirée des cartes, il choisit la difficulté (bleu, blanc, rouge).

Exemple de question : Qu’est-ce qu’un hypertexte ?

Bonne réponse = avance de 2 cases supplémentaires.

Mauvaise réponse = reste sur place.

Cases Défis : Si le joueur tombe sur une case “Défi”, il tire une carte Défi et doit accomplir un petit challenge.

Exemple de défi : Donne un exemple d’ergonomie cognitive dans un objet que tu utilises tous les jours.

Réussir le défi = avance de 3 cases supplémentaires.

Échouer = recule de 1 case.

Cases Neutres : Le joueur peut se reposer sur cette case, et rien ne se passe.

Fin du jeu : Le premier joueur à atteindre la dernière case du plateau gagne.

Exemples de questions

Canaux : Cite trois canaux de communication différents.

Médias : Qu’est-ce qu’un média ? Donne un exemple.

Hypertexte : Quelle est la différence entre un hypertexte et un hyperdocument ?

Ergonomie : Qu’est-ce que l’ergonomie cognitive ?

Navigation : Décris un exemple de navigation étoile.
